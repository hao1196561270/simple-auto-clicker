# 鼠标连点器（简易自动点击工具）

一个使用 Python + tkinter 制作的简单自动连点工具，支持：

- 自定义点击间隔（毫秒级）
- 同时支持左/中/右键（可多选）
- 可选同时按下指定键盘按键
- 两种启动方式：  
  - F6 按住即连点（松开即停止）  
  - F7 一键切换开/关

适合游戏刷图、重复点击、自动化测试等场景。

## 系统要求

- Windows 10 / 11（推荐）  
- Python 3.8 ~ 3.11  
- 需要管理员权限运行（部分游戏/软件会检测非管理员点击）

使用教程

启动程序后出现如下界面：
点击间隔：填入毫秒数（推荐 50~200ms，游戏反作弊通常不允许低于 30~50ms）
鼠标按键：勾选你想要自动点击的按键（可多选）
键盘按键：想同时自动按的键填在这里（例如 "z"、"space"、"1" 等），留空则不按键盘
工作模式：
F6 按住启用 → 按住 F6 就开始疯狂连点，松开立即停止
F7 切换启用 → 按一下 F7 开启/关闭（切换状态）


设置完成后点击 【启动】 按钮（也可以直接用快捷键）
使用快捷键控制：
F6 模式：按住 F6 → 连点，松开 → 停止
F7 模式：按 F7 → 开始/停止 循环切换

想完全停止程序 → 直接关闭窗口即可
安全提醒

仅限单机游戏或个人测试使用
大多数联网游戏（尤其是竞技类）严格禁止使用任何连点/宏工具
使用后果自负（包括但不限于：账号封禁、设备被检测等）
