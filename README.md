# 鼠标连点器（简易自动点击工具）

一个使用 Python + tkinter + pynput 实现的轻量级自动连点工具。

支持：
- 自定义点击间隔（毫秒级）
- 可同时勾选左键 / 中键 / 右键（多选）
- 可选同时自动按下单个键盘按键（例如 z、space、1、F 等）
- 两种常用快捷键模式：
  - **F6** 按住即连点（松开立即停止）
  - **F7** 一键切换开启/关闭

适合用于单机游戏重复操作、网页挂机、自动化测试等个人场景。

## 系统要求

- **操作系统**：Windows 10 / 11（推荐）  
  Linux 和 macOS 理论上可运行，但快捷键监听和权限表现可能不稳定
- **Python**：3.8 ~ 3.11
- 建议**以管理员身份**运行（很多游戏窗口需要）

## 快速开始
```markdown

### 1. 安装依赖

```bash
# 推荐先创建虚拟环境
python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # macOS / Linux

# 安装必要库

```

> 注意：`tkinter` 通常随 Python 一起安装。若缺少，请参考：
> - Windows：一般自带
> - Ubuntu/Debian：`sudo apt install python3-tk`
> - macOS：`brew install python-tk`

### 2. 保存主程序文件

将代码保存为 **`鼠标连点器.pyw`**（注意后缀是 `.pyw`）

`.pyw` 后缀运行时不会弹出黑色控制台窗口，体验更好。

### 3. 运行程序

- **双击** `鼠标连点器.pyw`  
  或
- 命令行运行：
  ```bash
  python 鼠标连点器.pyw
  ```

推荐**右键 → 以管理员身份运行**，兼容性最好。

## 使用说明

1. 启动后设置：
   - **点击间隔**：建议 50–200 ms（太快容易被游戏反作弊检测）
   - **鼠标按键**：勾选需要自动点击的按键（可多选）
   - **键盘按键**：输入单个按键（例如 `z`、`space`、`1`），留空则不模拟键盘
   - **工作模式**：
     - F6 按住启用 → 按住 F6 开始连点，松开停止
     - F7 切换启用 → 按一次 F7 开启/关闭

2. 点击界面「**启动**」按钮（或直接使用快捷键）

3. 控制方式总结：

| 模式       | 快捷键 | 操作方式                     | 适合场景             |
|------------|--------|------------------------------|----------------------|
| 按住式     | F6     | 按住 = 连点，松开 = 停止     | 需要精准控制时长     |
| 切换式     | F7     | 按一下 = 开/关 切换          | 长时间挂机           |

4. 完全退出 → 直接关闭窗口

## 常见问题与解决

| 现象                     | 可能原因                              | 解决方法                              |
|--------------------------|---------------------------------------|---------------------------------------|
| 快捷键没反应             | 程序未点「启动」 / 窗口失焦           | 先点击启动按钮，或保持窗口在前台      |
| 游戏里点击无效           | 缺少管理员权限                        | 右键 → 以管理员身份运行               |
| 键盘模拟不生效           | 输入了不支持的键 / 组合键             | 只支持单个普通键（a-z 0-9 space 等）  |
| 点击太快被检测/封号      | 间隔过小、过于规律                    | 调大间隔（≥100ms），或后续版本加随机  |
| 程序卡顿 / CPU 占用高    | 频率过高                              | 调大间隔，或减少同时点击的按键数量    |
